Colour & Shades ERD

ERD Description

Table - Primary Colour

  • This table is used to sort primary colours. It shows primary colour names and identifying numbers.
  • The ID attribute gives each color a unique number for identification.
  • The Primary Colour Name attribute gives color names.
  • This table has one primary key ID and it is used a foreign key in the Shades table.

Table - Shades

  • This table is used to sort colour shades and see what primary colours they connect with using the Primary Colour ID attribute.
  • The ID attribute gives each shade a unique number for identification.
  • Primary Colour ID attribute is used to connect primary colour to a shade.
  • Colour Name attribute is used to know what colour shade there is.
  • This table uses primary key Primary Colour ID from Primary Colour table as a foreign key to connect shade to a colour. Shades can have one primary colour and colours can have many shades.
